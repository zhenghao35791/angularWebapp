# angularWebapp
使用angularjs编写的简单项目，仿写拉钩招聘。
# 项目依赖
```javascript
"dependencies": {
    "angular": "^1.6.2",
    "angular-ui-router": "ui-router#^0.4.2",
    "angular-cookies": "^1.6.2",
    "angular-validation": "^1.4.3",
    "angular-animate": "^1.6.2"
  }，
  "devDependencies": {
    "gulp": "^3.9.1",
    "gulp-clean": "^0.3.2",
    "gulp-concat": "^2.6.1",
    "gulp-connect": "^5.0.0",
    "gulp-cssmin": "^0.1.7",
    "gulp-imagemin": "^3.1.1",
    "gulp-less": "^3.3.0",
    "gulp-load-plugins": "^1.5.0",
    "gulp-uglify": "^2.0.1",
    "open": "0.0.5"
  }
```
# 运行项目方法如下：
## install dependencies
npm install 
## install gulp
npm install -g gulp
### install gulp locally in the porject
npm install gulp
# run gulp, serve with hot reload at localhost:8080
npm run gulp
